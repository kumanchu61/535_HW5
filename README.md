# 535_HW5 Directory

This directory includes the source code, knitted documents, raw-data, and the RStudio project file.

## Folders

**Code**
Includes files: 

- *Homework 5 Code.Rmd*: This is the main R Markdown file that contains all code.

- *Homework-5-Code.pdf*: This is the knitted PDF version of the R Markdown file that shows the code and output.


**Project**
Includes files:

- *535_HW5.Rproj*: This is the RStudio project, which when opened will set the working directory and and load the project environment. 


**Raw_Data**

- *homicide-data.csv*: This is the raw data used in the code and the R Markdown.
